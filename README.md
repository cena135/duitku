# DuitKu — Pencatat Keuangan Harian

Aplikasi keuangan pribadi yang mengikuti **Apple Human Interface Guidelines (HIG)**. Offline-first, zero install, sangat customizable. Bisa dipakai sebagai web app, PWA, atau native Android APK (via Capacitor).

![DuitKu](https://img.shields.io/badge/version-2.0.0-34c759) ![Offline](https://img.shields.io/badge/offline-first-007aff) ![PWA](https://img.shields.io/badge/PWA-ready-af52de) ![Capacitor](https://img.shields.io/badge/Capacitor-Android-ff9500)

## Struktur

```
DuitKu/
├── www/                 ← web app (HTML/CSS/JS), self-contained
│   ├── index.html
│   ├── app.js
│   ├── styles.css
│   ├── manifest.json    ← PWA manifest
│   └── sw.js            ← service worker
├── android/             ← native Android project (generated by Capacitor)
├── capacitor.config.json
├── package.json         ← npm scripts
├── README.md
└── BUILD_ANDROID.md     ← guide build Android (PWA / PWABuilder / Capacitor)
```

## Quick start

```powershell
# Run as web app (browser):
npm run serve
# → http://localhost:8000

# Build Android APK (Capacitor sudah di-setup):
npm run android:build
# → APK keluar di android/app/build/outputs/apk/debug/app-debug.apk

# Install langsung ke HP Android yang dicolok USB (USB debug ON):
npm run android:install

# Setiap kali edit code di www/, sync ke Android:
npm run android:sync

# Buka project di Android Studio:
npm run android:open
```

## Fitur

### Pencatatan
- **Transaksi** — pemasukan & pengeluaran dengan 18 kategori default (Makanan, Transport, Tagihan, Belanja, Hiburan, Kesehatan, Pendidikan, Keluarga, dll), filter, search, navigasi per bulan
- **Langganan** — Netflix, Spotify, ChatGPT, Indihome, dll dengan auto-hitung total per bulan & per tahun. Reminder mau jatuh tempo. Tombol "Tandai Sudah Bayar" auto-catat ke transaksi
- **Budget** — limit pengeluaran per kategori dengan progress bar warna (hijau/oranye/merah). Alert saat over
- **Goal Tabungan** — DP rumah, HP baru, liburan dengan progress & deadline

### Tampilan (Apple HIG)
- **Large title** ala iOS yang shrink ke compact saat scroll
- **Inset grouped lists** (style Settings.app iOS)
- **iOS segmented control** untuk filter & tab
- **Modal sheet** ala iOS dengan handle grab + tombol Batal/Simpan
- **Tab bar** dengan blur background
- **Spring animations** halus, tap feedback ala iOS

### Customizable (semuanya bisa diubah di Pengaturan)
- **Tema** — Otomatis (ikut sistem) / Terang / Gelap
- **Warna Aksen** — 9 pilihan: Biru, Hijau, Indigo, Oranye, Pink, Ungu, Merah, Tosca, Kuning
- **Mata Uang** — IDR, USD, EUR, GBP, JPY, SGD, MYR, AUD, CNY
- **Kerapatan** — Nyaman / Padat
- **Awal Bulan** — Pilih tanggal mulai siklus bulanan (1, 5, 10, 15, 20, 25, 28) untuk yang gajiannya bukan tanggal 1
- **Tampilkan Desimal** — ya / tidak
- **Kategori** — Edit nama & emoji semua kategori (Pengeluaran, Pemasukan, Langganan), tambah / hapus / reset ke default
- **Getar saat simpan** — toggle haptic feedback

### Laporan
- Pie chart pengeluaran per kategori
- Bar chart 6 bulan terakhir (income vs expense)
- Top 5 pengeluaran bulan ini
- Rata-rata pengeluaran per hari

### Data
- **Export JSON** untuk backup
- **Import JSON** restore kapan saja
- **Reset semua data** (dengan double confirm)
- Semua tersimpan di `localStorage` HP-mu, **no cloud, no tracking**

## Android APK sudah ready! 📱

File `DuitKu-debug.apk` (3.8 MB) sudah di-build di root folder ini. Tinggal install:

### Install ke HP Android (sideload):

1. **Pindah file `DuitKu-debug.apk` ke HP** — via:
   - USB cable + drag ke folder Downloads
   - WhatsApp ke diri sendiri / send to nomor sendiri
   - Email attachment ke email sendiri
   - Google Drive / Dropbox upload, download di HP
2. Di HP, **buka file `DuitKu-debug.apk`** dari Files / Downloads
3. Muncul warning **"Install dari sumber tidak dikenal"** → tap **Settings** → enable untuk Files / app yang kamu pakai buka APK
4. Tap **Install** → DuitKu muncul di app drawer dengan icon DuitKu
5. Buka app → langsung jalan, full offline, data tersimpan di HP

### Install langsung via USB (kalau ada Android SDK + USB debugging ON):

```powershell
npm run android:install
```

### Build ulang APK setelah edit kode:

```powershell
# 1. Edit file apapun di www/
# 2. Sync ke Android project:
npm run android:sync
# 3. Build APK baru:
npm run android:build
# APK baru di: android/app/build/outputs/apk/debug/app-debug.apk
```

## "Local" vs "Online" — Pilih Cara Akses

App ini cuma sekumpulan file HTML/CSS/JS. Cara akses tergantung kebutuhan:

### Cara 1: Cuma di laptop (local development)
Buka `index.html` di browser → langsung jalan. Tapi service worker & PWA install tidak optimal kalau pakai `file://` URL.

### Cara 2: Server lokal (HP cuma akses kalau 1 WiFi)
```powershell
# Di folder DuitKu/, jalanin:
python -m http.server 8000
```
Cek IP laptop: `ipconfig` → IPv4 (misal `192.168.1.10`). Di HP (harus 1 WiFi sama laptop), buka `http://192.168.1.10:8000` di Chrome. Lalu **Menu → Add to Home Screen**.

**Limitasi:** Cuma jalan saat laptop nyala + 1 WiFi. Kalau di luar rumah / laptop mati → app tidak bisa dibuka pertama kali. Kalau sudah pernah dibuka dan jadi PWA, service worker cache, **bisa dibuka offline**.

### Cara 3 (RECOMMENDED): Deploy gratis ke internet — bisa diakses dari mana pun, kapan pun, siapa pun

Paling gampang pakai **Netlify Drop**:

1. Buka https://app.netlify.com/drop
2. Drag-and-drop folder `DuitKu/` ini ke halaman tersebut
3. Tunggu ~10 detik, dapat URL seperti `https://duitku-abcd.netlify.app`
4. Buka URL di HP kapan saja (dari WiFi mana saja, atau data seluler)
5. **Add to Home Screen** → jadi "app" forever, **offline-ready** karena ada service worker

Alternatif gratis lain:
- **Vercel** — https://vercel.com (drag folder, similar dengan Netlify)
- **GitHub Pages** — push ke repo public, enable Pages
- **Cloudflare Pages** — https://pages.cloudflare.com

**Privasi:** Walaupun deployed online, **data kamu tidak ke server**. Semua tersimpan di HP-mu via `localStorage`. URL-nya cuma serve file statis HTML/CSS/JS. Aman.

## Backup — JANGAN LUPA

Data ada di browser HP. Kalau clear browser data / HP rusak / uninstall app → data **hilang**.

**Lakukan rutin** (minimal sebulan sekali):
1. Pengaturan → **Export (Backup)**
2. File JSON kecil (~beberapa KB) ke-download
3. Simpan ke Google Drive / iCloud / WhatsApp ke diri sendiri

Restore: Pengaturan → **Import Backup** → pilih file JSON.

## Tech

- **Vanilla HTML / CSS / JS** — no framework, no build step, no dependencies (selain Chart.js via CDN)
- **Chart.js 4.x** untuk grafik
- **LocalStorage** untuk persistence
- **Service Worker** untuk offline + PWA install
- **Intl.NumberFormat** untuk format mata uang & angka per locale
- **CSS custom properties** untuk theme + accent yang dinamis

## Customize Lebih Lanjut (untuk yang bisa coding sedikit)

- **Tambah mata uang baru:** Edit `app.js` di constant `CURRENCIES`
- **Tambah warna aksen baru:** Edit `ACCENTS` di `app.js` + tambah `.accent-swatch[data-c="..."]` di `styles.css`
- **Ganti emoji palette di category editor:** Edit `EMOJI_PALETTE` di `app.js`
- **Custom default category:** Edit `DEFAULT_CATEGORIES` di `app.js` (atau bisa di-edit langsung dari UI di Pengaturan → Kategori)
- **Ganti font:** Edit `body { font-family: ... }` di `styles.css`
